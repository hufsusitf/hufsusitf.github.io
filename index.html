Windows XP 사용자에 대한 안내 / Notice for Windows XP User

www.usitf.kr

한국어 (Korean)

Windows XP 환경에서의 통번역연합회 홈페이지 이용에 대한 안내

현재 홈페이지를 운영중인 서버에서는 SNI (Server Name Indication)을 사용하여 도메인 주소에 별도의 TLS 포트를 지정하지 않고 HTTPS 프로토콜로 연결하고 있습니다.

다만 이 기술은 Windows Vista부터 운영체제 자체의 보안 기능에 의해 지원되고 있으며, Windows XP 또는 그 이하의 운영체제에 설치된 Internet Explorer에서는 지원되지 않고 있어 홈페이지 접속 시 인증서 오류를 표시하기 때문에 Windows XP에 설치된 Internet Explorer 에서는 정상적으로 이용이 불가능합니다.

따라서 최신 버전의 운영체제 또는 Google Chrome이나 Mozilla Firefox를 사용하여 접속해 주시기 바랍니다. 이용에 불편을 드려 죄송합니다.

English

Notice : Visiting USITF Web Site on Windows XP Environment

Currently, the server runs our web site uses SNI (Server Name Indication) to connect via HTTPS protocol without indicating any specific TLS port on the domain name.

Unfortunately, this technology is only suppored on Windows Vista or higher by their own security features inside of the operating system. So it is NOT supported by Internet Explorer on Windows XP or lower.

So please use the latest operating system or browse this web site with Google Chrome or Mozilla Firefox. Sorry for the inconvienence.
